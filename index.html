🖥️ TechStore API

REST API Sistem Kasir Toko HP dan Laptop


📚 Kelompok 5 🎓 Integrasi Sistem ⚡ Node.js + MySQL

📌 Pendahuluan

TechStore API adalah REST API yang dibangun menggunakan Node.js dan Express.js sebagai backend sistem kasir toko HP dan Laptop. API ini mengelola data produk, customer, transaksi penjualan, dan laporan secara lengkap dengan koneksi database MySQL (XAMPP).

🔒 Middleware

📝 Logger

Mencatat setiap request yang masuk beserta waktu dan method

✅ Validator

Validasi input request body sebelum diproses controller

❌ Error Handler

Menangani error secara global dan mengembalikan response yang rapi

🔍 Not Found

Menangani route yang tidak ditemukan dengan pesan yang jelas
