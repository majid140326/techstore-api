🖥️ TechStore API

Dokumentasi REST API Sistem Kasir & Manajemen Inventori Toko HP dan Laptop

📚 Kelompok 5 🎓 Integrasi Sistem ⚡ Node.js + MySQL

📌 Pendahuluan

TechStore API adalah platform RESTful API backend yang dirancang khusus untuk mendukung operasional sistem kasir toko ritel HP dan Laptop. API ini mampu mengotomatisasi pendataan stok barang, pengelolaan data pelanggan secara terpusat, validasi transaksi penjualan seketika, serta menghasilkan rekapitulasi data laporan bisnis.
