🚀 Cara Menjalankan Project

1. Install Dependencies

npm install

2. Buat Database

Buka phpMyAdmin di http://localhost/phpmyadmin dan jalankan SQL berikut:

CREATE DATABASE techstore;

USE techstore;

CREATE TABLE products (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nama VARCHAR(100) NOT NULL,
    kategori VARCHAR(50) NOT NULL,
    harga INT NOT NULL,
    stok INT NOT NULL
);

CREATE TABLE customers (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nama VARCHAR(100) NOT NULL,
    telepon VARCHAR(20) NOT NULL,
    alamat TEXT NOT NULL
);

CREATE TABLE transactions (
    id INT AUTO_INCREMENT PRIMARY KEY,
    customer_id INT NOT NULL,
    total_harga INT NOT NULL,
    tanggal DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (customer_id) REFERENCES customers(id)
);

CREATE TABLE transaction_items (
    id INT AUTO_INCREMENT PRIMARY KEY,
    transaction_id INT NOT NULL,
    product_id INT NOT NULL,
    nama VARCHAR(100) NOT NULL,
    harga INT NOT NULL,
    jumlah INT NOT NULL,
    subtotal INT NOT NULL,
    FOREIGN KEY (transaction_id) REFERENCES transactions(id),
    FOREIGN KEY (product_id) REFERENCES products(id)
);

3. Jalankan Server

node server.js

Server berjalan di http://localhost:3000

⚠️ Catatan Penting Penggunaan :id

Beberapa endpoint menggunakan :id di URL. Ganti :id dengan angka ID yang sesuai.


Contoh:

  • http://localhost:3000/api/v1/products/:id → ganti :id dengan angka, contoh: http://localhost:3000/api/v1/products/1
  • http://localhost:3000/api/v1/customers/:id → ganti :id dengan angka, contoh: http://localhost:3000/api/v1/customers/2
  • http://localhost:3000/api/v1/transactions/:id → ganti :id dengan angka, contoh: http://localhost:3000/api/v1/transactions/1

Cara mengetahui ID yang tersedia:

  1. Jalankan GET semua data terlebih dahulu.
  2. Lihat nilai "id" pada data yang muncul.
  3. Gunakan nilai "id" tersebut untuk mengganti :id di URL.

Contoh: Jalankan GET http://localhost:3000/api/v1/products terlebih dahulu

[
    { "id": 1, "nama": "ASUS TUF A15", ... },
    { "id": 2, "nama": "iPhone 15", ... }
]

Maka untuk melihat detail produk pertama gunakan: http://localhost:3000/api/v1/products/1

📮 Panduan Penggunaan CRUD di Postman

Langkah-Langkah Umum Testing API:

  1. Buka aplikasi Postman.
  2. Klik tombol + New Request (atau tanda tambah di tab atas).
  3. Pilih Method HTTP pada *dropdown* sebelah kiri (GET, POST, PUT, DELETE).
  4. Masukkan Endpoint URL di kolom panjang (contoh: http://localhost:3000/api/v1/products).
  5. Untuk metode POST dan PUT (Tambah/Update Data):
    • Pilih tab Body tepat di bawah kolom URL.
    • Pilih opsi raw.
    • Ubah dropdown *Text* di pojok kanan menjadi JSON.
    • Masukkan format/struktur kode JSON ke dalam kotak teks yang tersedia.
  6. Untuk metode GET dan DELETE: Tidak perlu mengisi tab Body. Biarkan kosong.
  7. Klik tombol biru Send.
  8. Hasil data (Response) akan langsung muncul di kotak bagian bawah aplikasi Postman.
