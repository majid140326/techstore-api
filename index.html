📌 Pendahuluan

🚀 Cara Penggunaan di Postman

  1. Pastikan server API sudah berjalan di lokal (npm run dev atau node server.js).
  2. Buka aplikasi Postman dan buat Workspace/Collection baru.
  3. Klik tombol + New Request.
  4. Pilih metode HTTP yang sesuai (GET, POST, PUT, DELETE) pada dropdown di sebelah kiri kolom URL.
  5. Masukkan Base URL + Endpoint ke kolom URL. Contoh: http://localhost:3000/api/v1/products
  6. Khusus untuk metode POST dan PUT (Tambah/Update Data):
    • Pilih tab Body di bawah kolom URL.
    • Pilih opsi raw.
    • Ubah format dari Text menjadi JSON di dropdown sebelah kanan.
    • Masukkan format JSON (Payload) sesuai dokumentasi di bawah.
  7. Klik tombol Send (warna biru) di sebelah kanan.
  8. Lihat hasil balasan dari server di bagian bawah pada kotak Response (Body).

📦 1. Products (Produk)

GET/api/v1/productsAmbil semua produk

Hasil Response (200 OK):

[
  { "id": 1, "nama": "ASUS TUF A15", "kategori": "Laptop", "harga": 12000000, "stok": 10 },
  { "id": 2, "nama": "iPhone 15", "kategori": "HP", "harga": 15000000, "stok": 8 }
]
POST/api/v1/productsTambah produk baru

Request Body (JSON):

{
  "nama": "MacBook Air M2",
  "kategori": "Laptop",
  "harga": 18000000,
  "stok": 5
}

Hasil Response (201 Created):

{
  "message": "Produk berhasil ditambahkan",
  "insertId": 3
}
PUT/api/v1/products/:idUpdate data produk

Request Body (JSON):

{
  "nama": "ASUS TUF A15 (Updated)",
  "harga": 11500000,
  "stok": 15
}

Hasil Response (200 OK):

{
  "message": "Produk berhasil diupdate"
}
DELETE/api/v1/products/:idHapus produk

Hasil Response (200 OK):

{
  "message": "Produk berhasil dihapus"
}

📝 Kesimpulan

TechStore API berhasil dibangun dengan fitur fungsional lengkap yang memenuhi standar industri meliputi:


  • ✅ Implementasi CRUD Lengkap untuk data Products, Customers, dan Transactions.
  • ✅ Arsitektur Keamanan melalui Middleware Logger, Validator, dan Error Handler.
  • ✅ Relasi Database tersusun rapi menggunakan MySQL (XAMPP).
  • ✅ Laporan Intelijen Bisnis mencakup Omset Penjualan, Top Produk, dan Sisa Stok (Warning).
  • ✅ Telah Diuji Sepenuhnya menggunakan Postman dan berjalan tanpa kendala (*bug-free*).
